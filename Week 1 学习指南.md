## 核心概念

### 必须理解的概念

1. **Grid/Block/Thread层级**
```
Grid (整个计算空间)
└── Block (线程组)
    └── Thread (单个线程)
```

2. **索引计算**
```cpp
int idx = blockIdx.x * blockDim.x + threadIdx.x;
```

3. **内存模型**
```
Host Memory (CPU)  ←→  Device Memory (GPU)
     malloc              cudaMalloc
     free                cudaFree
```

4. **Kernel启动**
```cpp
kernel<<<blocks, threads>>>(args);
```

5. **边界检查**
```cpp
if (idx < n) {
    // 处理数据
}
```

## 常见问题

### Q1: 编译错误 - 找不到cuda_runtime.h
**A:** 检查CUDA是否正确安装，环境变量是否配置

### Q2: 运行时错误 - GPU上没有输出
**A:** 需要调用 cudaDeviceSynchronize()

### Q3: 结果不对
**A:** 检查边界条件，检查索引计算

### Q4: 编译很慢
**A:** 第一次编译会慢，之后会快很多
